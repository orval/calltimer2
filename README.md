# Call Timer

An 45-minute call countdown timer.

Enter a call start time and the clock will count down until then. Once the call starts, it counts up to 45 minutes.

The clock will show different colours based on the relative time as follows:

* White - there are more than 3 minutes to go until the call starts
* Red - there are less than 3 minutes to go
* Green - call is in progress
* Amber - less than 5 minutes remaining
* Red again - over 45 minutes from start
